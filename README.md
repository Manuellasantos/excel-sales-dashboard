# excel-sales-dashboard
Este projeto apresenta um fluxo completo de análise em Excel utilizando dados fictícios de vendas.
O conjunto de dados foi organizado em tabelas (Produtos, Clientes, Representantes e Registros de Vendas), analisado por meio de tabelas dinâmicas e transformado em um dashboard interativo exibindo KPIs e insights de desempenho.

O objetivo deste projeto é praticar habilidades essenciais de limpeza de dados, análise exploratória, visualização e comunicação de insights, usando o Excel como ferramenta de Business Intelligence.
